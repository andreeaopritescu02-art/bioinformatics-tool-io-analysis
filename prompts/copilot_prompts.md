# Coding Agent Prompts

These prompts will be used to analyze bioinformatics tool repositories.

## 1. Tool overview

Describe the purpose of this bioinformatics tool and its main functionality.

## 2. Input files

What input files does this tool accept?

For each input:
- provide the file format;
- describe the data type;
- provide evidence from the repository (README, documentation, source code, examples).

## 3. Output files

What output files are generated by this tool?

For each output:
- provide the file format;
- describe the data type;
- provide evidence from the repository.

## 4. Supported formats

Search the repository and identify all supported input and output file formats.

Mention where this information was found.

## 5. Source code analysis

Search the source code for:
- file parsers;
- file readers;
- file writers;
- format-specific implementations.

Explain what formats they support.

## 6. EDAM mapping

Suggest suitable EDAM ontology terms for:
- input data types;
- output data types;
- file formats.

Explain the reasoning behind each suggestion.

## 7. Confidence and limitations

Identify any uncertain information or assumptions made during the analysis.
