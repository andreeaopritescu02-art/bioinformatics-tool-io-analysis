# Project 4 – Using Coding Agents to Find Input and Output Formats of Bioinformatics Tools

## Overview

This project investigates whether AI coding agents (e.g., GitHub Copilot) can assist in identifying the input and output data formats of bioinformatics software by exploring their public source code repositories.

The extracted information is manually verified and mapped to EDAM data type and file format terms, following the metadata standards used by bio.tools.

## Objectives

- Explore public bioinformatics software repositories.
- Identify supported input and output files.
- Collect evidence from documentation and source code.
- Map the identified formats to EDAM ontology terms.
- Evaluate the accuracy of AI-generated suggestions.
